DaLinkOfeeder
=============

Terraria DaLinkO feeder mod compatible with TAPI r11

This mod currently transmits the following stats to DaLinkO

player.statLife (int) - player's current health level
player.statLifeMax (int) - player's current max possible health
player.statMana (int) - player's current mana level
player.statManaMax (int) - player's current max possible mana
player.breath (int) - player's current amount of breath
player.breathMax (int) - player's current max possible breath
player.dead (bool) - player's current live or dead status
